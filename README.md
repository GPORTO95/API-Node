## Projeto API Node JS

Projeto Open-Source para CRUD de produtos.

## Tecnologias implementadas:
- Node JS
- MongDB
- Autenticação
- Autorização
- Envio de e-mail

## Arquitetura:
- Repository
- Rotas
- Controllers
- Models
- Validações
- Async/Await

## Requirements
- Node.js v15+
- MongoDB Atlas

## Running
- Install dependencies - npm i
- Build typescript - npm run build
- Run project - npm start
- Go to swagger page - localhost:3000/
